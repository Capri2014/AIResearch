# Docs

This folder contains documentation, research notes, surveys, and digests.

## Folder Structure

| Folder | Purpose | Content |
|--------|---------|---------|
| **`papers/`** | Structured paper series | Related papers analyzed together (evolution of architecture) |
| **`surveys/`** | Broad topic surveys | Many papers on one theme (50+ papers on VLA, CoT, etc.) |
| **`digests/`** | Deep-dive single papers | Detailed analysis of ONE paper |
| **`learning/`** | Learning notes | BERT, JEPA, SSL concepts explained |
| **`compare/`** | Comparisons | Side-by-side analysis of approaches |

## Examples

- `digests/2026-02-13-drive-jepa.md` → Deep dive into ONE paper
- `surveys/2026-02-16-vla-world-model-2025-survey.md` → Survey of 50+ papers
- `papers/series-architecture-papers.md` → 3 related papers (GAIA-1 → UniAD → Drive-JEPA)

## Contributing

- Keep docs small and link out to sources
- Prefer one topic per file
- Use date format: `YYYY-MM-DD-topic.md`
