# ClawBot activity log

This folder tracks what **ClawBot (OpenClaw)** did in this repo over time.

## Workflow
- **One PR per day** (California time)
- **One commit per day** (small, focused)
- Each daily PR includes a matching daily log entry: `clawbot/daily/YYYY-MM-DD.md`

## Files
- `clawbot/STATUS.md` — current focus + next steps + blockers
- `clawbot/daily/` — one file per day (what changed + why + what to review)
