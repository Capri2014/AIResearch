## Pipeline PR #3: Waypoint Visualization Module (This PR)

### Created: `sim/driving/carla_srunner/waypoint_visualizer.py`

- **VisualizationConfig**: Configuration dataclass for all visualization options
- Supports both 2D BEV image rendering and CARLA 3D world visualization
- Features:

  - Waypoint numbering and heading arrows
  - Reference path visualization
  - Speed prediction display
  - CARLA 3D debug drawing

- Factory function `create_visualizer()`

### Updated: `sim/driving/carla_srunner/policy_wrapper.py`
- Added import for waypoint_visualizer module
- Added `WAYPOINT_VISUALIZER_AVAILABLE` flag

### Usage:
```python
from sim.driving.carla_srunner.waypoint_visualizer import create_visualizer

# Create visualizer
viz = create_visualizer(
    image_width=400,
    image_height=400,
    meters_per_pixel=0.5
)

# Create BEV image with waypoints
image = viz.create_bev_image(
    waypoints=waypoints,  # Shape (N, 2) or (N, 3) in vehicle frame
    vehicle_heading=0.0,
    predicted_speed=8.0,
    reference_path=route_path,  # Optional
)

# In CARLA:
viz.draw_waypoints_carla(carla_world, waypoints, vehicle_transform)
viz.draw_trajectory_carla(carla_world, trajectory)
```

## Pipeline PR #2: Waypoint Inference + CarlaWaypointAgent

### Created: `sim/driving/carla_srunner/waypoint_inference.py`
- **WaypointInference**: Loads trained BC checkpoint, predicts waypoints from BEV
- Batch inference support
- Speed prediction
- Waypoint coordinate transformation (vehicle frame → global frame)
- Factory function `create_inference()`

### Created: `sim/driving/carla_srunner/carla_waypoint_agent.py`
- **CarlaWaypointAgent**: End-to-end driving agent
- Integrates WaypointInference + WaypointTrackingController
- `compute_control(bev_image, vehicle)`: Full perception → control pipeline
- `compute_control_from_waypoints()`: Direct waypoint input for RL/external policies
- MockVehicle for testing without CARLA
- Factory function `create_agent()`

### Updated: `sim/driving/carla_srunner/policy_wrapper.py`
- Added imports for new modules (CARLA_AGENT_AVAILABLE, WAYPOINT_INFERENCE_AVAILABLE)
- Added `use_full_agent` config option

### Architecture:
```
BEV Image
    ↓
WaypointInference (BC checkpoint)
    ↓
waypoints [num_waypoints, 2]
    ↓
CarlaWaypointAgent
    ├── WaypointTrackingController
    │   ├── Pure Pursuit → steering
    │   ├── Curvature → target speed
    │   └── PID → throttle/brake
    ↓
CARLA VehicleControl
```

### Usage:
```python
from sim.driving.carla_srunner.carla_waypoint_agent import create_agent

# Create agent
agent = create_agent(
    checkpoint_path="out/waypoint_bc/run_XXXX/best.pt",
    vehicle_type="tesla_model3"
)

# In simulation loop:
control = agent.compute_control(bev_image, vehicle)
# Returns: {"throttle": 0.5, "steer": 0.1, "brake": 0.0, ...}
```
