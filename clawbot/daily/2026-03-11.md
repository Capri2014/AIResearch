# Daily Notes - 2026-03-11

## Pipeline PR #1: Waypoint Tracking Controller for CARLA

### Created: `sim/driving/carla_srunner/waypoint_controller.py`
- **WaypointTrackingController**: Sophisticated waypoint-to-control converter
- **Pure pursuit** steering controller with dynamic lookahead
- **Speed profiling** based on path curvature
- **PID speed controller** for smooth acceleration/deceleration
- **Obstacle proximity braking** for safety
- **Steering smoothing** for less jerky control

### Features:
- Dynamic lookahead based on speed (1.0s lookahead time + 2m min)
- Curvature-based speed reduction (reduces speed on turns)
- PID controller with anti-windup for speed tracking
- Multiple vehicle presets (Tesla Model 3, Ford Escape, generic)
- CARLA VehicleControl format output

### Architecture:
```
Waypoints (from policy)
    ↓
WaypointTrackingController
    ├── Pure Pursuit → steering
    ├── Curvature Analysis → target speed  
    ├── PID Speed Controller → throttle/brake
    └── Smoothing → smooth control
    ↓
CARLA VehicleControl (throttle, steer, brake)
```

### Updated: `sim/driving/carla_srunner/policy_wrapper.py`
- Added optional integration with WaypointTrackingController
- New config options: `use_advanced_controller`, `vehicle_type`
- Falls back to simple controller if advanced not available

### Usage:
```python
from sim.driving.carla_srunner.waypoint_controller import create_controller

# Create controller for Tesla Model 3
controller = create_controller("tesla_model3")

# Each control step:
control = controller.get_control_as_carla(waypoints, current_speed, dt)
# Returns: {"throttle": 0.5, "steer": 0.1, "brake": 0.0, "hand_brake": False, "reverse": False}
```

### Test Results:
- Straight path (8 m/s): throttle=1.0, steer=0.0
- Curved path (8 m/s): throttle=1.0, steer=0.025, target_speed=13.37 m/s
