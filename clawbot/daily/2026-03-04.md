### Architecture
```
SceneTransformerEncoder:
├── AgentHistoryEncoder (temporal attention)
├── MapPolylineEncoder (point → polyline pooling)
├── Agent-to-Map Cross Attention (num_layers=3)
├── Agent-to-Agent Self Attention
└── Output: agent_embeddings + scene_embedding

SceneTransformerWithWaypointHead:
├── SceneTransformerEncoder
└── ProposalWaypointHead (K=5 modes) or regression head
```

### Key Components
- **Temporal attention**: Multi-head self-attention over agent history
- **Map encoding**: Point-level → polyline-level mean pooling
- **Cross-attention**: Agents attend to relevant map elements
- **Agent interaction**: Self-attention for agent-agent interactions

### Usage Example
```python
from training.sft.scene_encoder import SceneTransformerWithWaypointHead, SceneEncoderConfig

config = SceneEncoderConfig(
    num_agents=32,
    num_map_points=256,
    num_history=20,
    hidden_dim=256,
    output_dim=256,
)
model = SceneTransformerWithWaypointHead(config, use_proposal_head=True)

# (B, A, T, 7) = batch, agents, timesteps, features
# (B, P, M, 3) = batch, polylines, points per polyline, (x, y, is_endpoint)
outputs = model(agent_history, map_polylines, agent_masks, polyline_masks)
# outputs['proposals']: (B, K, H, 2)
# outputs['scores']: (B, K)
```
