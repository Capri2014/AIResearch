## Pipeline PR #1 (from earlier today): Scene Transformer Encoder Implementation

### Completed
- **Created `training/sft/scene_encoder.py`**
  - SceneTransformerEncoder: Core encoder with temporal attention for agents
  - MapPolylineEncoder: Encodes map polylines (roads, lanes)
  - CrossAttentionLayer: Agent-to-map cross attention
  - AgentHistoryEncoder: Temporal self-attention over trajectory history
  - SceneTransformerWithWaypointHead: End-to-end model with waypoint prediction
  - Supports proposal head (K modes) or simple regression

### Architecture
```
SceneTransformerEncoder:
├── AgentHistoryEncoder (temporal attention)
├── MapPolylineEncoder (point → polyline pooling)
├── Agent-to-Map Cross Attention (num_layers=3)
├── Agent-to-Agent Self Attention
└── Output: agent_embeddings + scene_embedding

SceneTransformerWithWaypointHead:
├── SceneTransformerEncoder
└── ProposalWaypointHead (K=5 modes) or regression head
```
