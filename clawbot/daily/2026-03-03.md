## Pipeline PR #1: Enhanced RL Training with Dense Rewards

### Problem
Previous RL training underperformed SFT after short runs (20 episodes):
- RL Success Rate: 0%
- SFT Success Rate: 40%

### Solution
Added dense reward signals for faster RL learning:

1. **DenseRewardWaypointEnv** - Enhanced environment with:
   - Progress rewards (step-by-step distance reduction)
   - Waypoint tracking rewards (how well following SFT waypoints)
   - Smooth control rewards (penalize jerky actions)
   - Speed efficiency rewards

2. **ImprovedDeltaHead** - Better delta architecture:
   - Layer normalization for stability
   - Multiple MLP layers (3 hidden layers)
   - Orthogonal weight initialization
   - Dropout for regularization

3. **EnhancedPPOAgent** - Full PPO training with GAE

### Testing

```bash
python -m training.rl.enhanced_rl_dense_train --smoke --verbose
```

Results (20 episodes):
- **Success Rate: 60%** (vs 0% before)
- Avg Return: 20.66 ± 5.79
- Eval Return: 23.16
