## Pipeline PR #3: Unified Training Pipeline for Latent Dynamics + Reasoning

### Summary
Added unified training pipeline that integrates latent dynamics model with reasoning trace decoder for combined model-based RL and explainability.

### Changes
- `training/rl/unified_training.py`: New file with:
  - **UnifiedTrainingConfig**: Configuration for combined training
  - **UnifiedWaypointDataset**: Dataset handling for both latent and reasoning inputs
  - **UnifiedTrainingState**: Main training loop with:
    - Latent dynamics components (encoder, dynamics, reward, uncertainty)
    - Reasoning model (WaypointReasoningModel)
    - Alternating optimization between latent and reasoning components
    - Imagine rollouts for model-based RL
  
- `training/rl/reasoning_trace_decoder.py`: Copied from AIResearch-repo submodule

### Architecture
- **Latent Dynamics**: Encode obs → latent state → predict next latent → reward/uncertainty
- **Reasoning**: Encode state + trajectory → predict waypoints + reasoning traces + confidence + risk
- **Combined Loss**: dynamics + reward + uncertainty + waypoint + reasoning weights

### Test Results
- dynamics_loss: 0.857
- reward_loss: 2.928
- uncertainty_loss: 2.764
- reasoning_loss: 0.100
- total_loss: 3.850

## Pipeline PR #2: Reasoning Trace Decoder for Interpretable Driving Decisions

### Summary
Added reasoning trace decoder module for explainable waypoint predictions, complementing the latent dynamics model with interpretability.

### Changes
- `training/rl/reasoning_trace_decoder.py`: New file with:
  - **SceneEncoder**: Encodes state and trajectory context
  - **ReasoningDecoder**: Generates reasoning tokens explaining decisions
  - **WaypointReasoningModel**: Combined prediction + explanation model
  - **ReasoningLoss**: Multi-task loss (waypoint + confidence + risk + reasoning)

- Cross-attention for reasoning token generation
- Predict waypoints alongside reasoning traces
- Enables human-interpretable decision traces for safety analysis
