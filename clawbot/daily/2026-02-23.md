## Pipeline PR #1: Latent Dynamics Model for Model-Based RL

### Summary
Added latent dynamics model for model-based RL planning, following the GAIA-2 style approach.

### Changes
- `training/rl/latent_dynamics_model.py`: New file with:
  - **LatentEncoder**: VAE-style encoder (mean + logvar)
  - **LatentDynamicsModel**: Learns transitions in latent space
  - **RewardPredictor**: Predicts rewards from latent + action
  - **UncertaintyModel**: Ensemble-based epistemic uncertainty (5 heads)
  - **LatentDynamicsRL**: Full model with imagined rollouts
  - **ModelBasedPPOAgent**: PPO + latent dynamics integration

### Architecture
- Encode obs → latent state
- Predict next latent via dynamics model
- Use uncertainty for risk-aware planning
- Enables imagined rollouts for sample-efficient learning

### Test Results

- reward_loss: 0.377
- uncertainty_loss: 0.043

## Pipeline PR #2: Reasoning Trace Decoder for Interpretable Driving Decisions

### Summary
Added reasoning trace decoder module for explainable waypoint predictions, complementing the latent dynamics model with interpretability.

### Changes
- `training/rl/reasoning_trace_decoder.py`: New file with:
  - **SceneEncoder**: Encodes state and trajectory context
  - **ReasoningDecoder**: Generates reasoning tokens explaining decisions
  - **WaypointReasoningModel**: Combined prediction + explanation model
  - **ReasoningLoss**: Multi-task loss (waypoint + confidence + risk + reasoning)
  - **ReasoningTrace**: Structured trace with observation, prediction, plan, confidence, risk

### Architecture
- Encode current state + trajectory history
- Cross-attention for reasoning token generation
- Predict waypoints alongside reasoning traces
- Enables human-interpretable decision traces for safety analysis

### Test Results
- waypoint_loss: 0.9746
- confidence_loss: 0.6987
- risk_loss: 0.7169
- reasoning_loss: 2.9972
- total_loss: 1.2660
