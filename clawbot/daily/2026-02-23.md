## Pipeline PR #1: Latent Dynamics Model for Model-Based RL

### Summary
Added latent dynamics model for model-based RL planning, following the GAIA-2 style approach.

### Changes
- `training/rl/latent_dynamics_model.py`: New file with:
  - **LatentEncoder**: VAE-style encoder (mean + logvar)
  - **LatentDynamicsModel**: Learns transitions in latent space
  - **RewardPredictor**: Predicts rewards from latent + action
  - **UncertaintyModel**: Ensemble-based epistemic uncertainty (5 heads)
  - **LatentDynamicsRL**: Full model with imagined rollouts
  - **ModelBasedPPOAgent**: PPO + latent dynamics integration

### Architecture
- Encode obs → latent state
- Predict next latent via dynamics model
- Use uncertainty for risk-aware planning
- Enables imagined rollouts for sample-efficient learning

### Test Results
- dynamics_loss: 0.027
- reward_loss: 0.377
- uncertainty_loss: 0.043
