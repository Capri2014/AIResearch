### 1. Scenario Configuration Module
- **Created:** `sim/driving/carla_srunner/scenario_config.py`

**10 Standard Scenarios:**
- straight_clear, straight_cloudy, straight_night, straight_rain
- turn_left_clear, turn_right_clear
- lane_change_clear, merge_clear
- straight_fog, straight_sunset

**Scenario Suites:**
- smoke (2): straight_clear, turn_left_clear
- quick (3): straight_clear, straight_cloudy, turn_left_clear
- full (8): all major scenarios
- adverse (3): rain, fog, sunset
- night (1): straight_night

**Helper Functions:**
- get_scenario(id): Get specific scenario
- get_scenario_suite(name): Get predefined suite
- get_scenarios_by_tag(tag): Filter by tag
- export_suite_to_json(): Export to JSON

## Architecture
```
                    ┌─────────────────────────┐
                    │   ScenarioConfig        │
                    │   - id, name, type      │
                    │   - weather, map        │
                    │   - route, timeout      │
                    └───────────┬─────────────┘
                                │
        ┌───────────────────────┼───────────────────────┐
        │                       │                       │
        ▼                       ▼                       ▼
┌───────────────┐     ┌───────────────┐     ┌───────────────┐
│ WeatherConfig │     │ RouteDef      │     │ EvalCriteria │
│ - preset      │     │ - waypoints   │     │ - timeout    │
│ - clouds      │     │ - distance    │     │ - threshold  │
│ - rain        │     │ - start/end   │     │ - max_collis │
└───────────────┘     └───────────────┘     └───────────────┘
```

## Usage
```python
from sim.driving.carla_srunner.scenario_config import (
    get_scenario, get_scenario_suite, list_available_scenarios
)

# Get specific scenario
scenario = get_scenario("straight_clear")
print(f"{scenario.name}, {scenario.map.value}")

# Get full evaluation suite
scenarios = get_scenario_suite("full")

# Export to JSON for ScenarioRunner
export_suite_to_json("full", Path("out/scenarios/full.json"))
```
