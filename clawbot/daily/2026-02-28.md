## Pipeline PR #1: GRPO Training for Residual Delta Waypoint Learning (2026-02-28)

### Summary
Added a new GRPO training script that combines Group Relative Policy Optimization with residual delta learning for waypoint prediction after SFT.

### Changes

**training/rl/train_grpo_delta_waypoint.py** (NEW - 549 lines):
- **GRPODeltaConfig**: Configuration dataclass for SFT checkpoint loading + GRPO training
- **GRPODeltaTrainer**: Complete training loop with group-relative advantages
  - Loads frozen SFT model for waypoint prediction
  - Trains delta_head using GRPO algorithm
  - Supports multi-scenario training via MultiScenarioEnv
  - Checkpoint saving and metrics logging
- **Group-relative advantages**: Normalize advantages within groups of related scenarios
- **CLI args**: --sft-checkpoint, --episodes, --scenarios, --group-size, --lr, --smoke

### Architecture
```
final_waypoints = sft_waypoints + delta_head(state)
```

### GRPO Benefits
- No value function needed (simpler, more stable)
- Group-based advantage estimation for related scenarios
- Sample multiple actions per state for better exploration

### Usage
```bash
# Train with SFT checkpoint
python training/rl/train_grpo_delta_waypoint.py --sft-checkpoint path/to/sft_model.pt --episodes 500

# Multi-scenario training
python training/rl/train_grpo_delta_waypoint.py --sft-checkpoint path/to/sft_model.pt --scenarios highway urban

# Smoke test
python training/rl/train_grpo_delta_waypoint.py --smoke --episodes 50
```

## Pipeline PR #4: Waypoint BC Training with Integrated Evaluation Metrics (2026-02-28)

### Summary
Added waypoint BC training script with ADE/FDE metrics computed during training for checkpoint selection. Follows Evaluation-First Design principle from MEMORY.md.

### Changes

**training/rl/train_waypoint_bc_with_metrics.py** (NEW - ~400 lines):
- **WaypointDataset**: Load waymo episodes from npz files
- **ToyWaypointDataset**: Toy dataset for testing without real data
- **compute_ade()**: Average Displacement Error metric
- **compute_fde()**: Final Displacement Error metric
- **evaluate_model()**: Full evaluation with loss, ADE, FDE
- **train_epoch()**: Training with gradient clipping
- **save_checkpoint()**: Save latest + best_ade checkpoint
- Integrated eval every N epochs for checkpoint selection
- Outputs: train_metrics.json with full history
