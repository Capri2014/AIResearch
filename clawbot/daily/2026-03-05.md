## Pipeline PR #2: Waymo Episode Data Loader + SSL Pretrain

### Completed
- **Created `training/sft/dataloader_waymo.py`** (330+ lines)
  - WaymoMotionDataset: PyTorch dataset for Waymo Open Motion Dataset
  - Loads TFRecords and extracts agent trajectories + map polylines
  - WaymoToWaypointBCAdapter: Converts Waymo samples to waypoint BC format
  - create_waymo_dataloader(): Factory function for DataLoader creation
  - Supports historical_steps (20 = 2s @ 10Hz) and future_steps (30 = 3s)
  - Tested: import OK

- **Created `training/sft/train_waymo_ssl.py`** (470+ lines)
  - WaymoSSLModel: SSL-pretrained Scene Transformer
  - MaskedTrajectoryPredictor: Reconstructs masked agent timesteps
  - ContrastiveMapAgentHead: Map-agent alignment via contrastive learning
  - FuturePredictionHead: Auxiliary future trajectory prediction
  - train_epoch() / evaluate(): Training loop with SSL objectives
  - Supports --resume for checkpoint continuation
  - Tested: import OK

### Architecture
```
WaymoMotionDataset:
├── Loads TFRecords (agent history + future + map)
├── Returns tensors:
│   ├── agent_history: (A, T, 7) [x, y, z, vx, vy, yaw, type]
│   ├── agent_future: (A, H, 2) [x, y]
│   └── map_polylines: (P, M, 3) [x, y, is_endpoint]

WaymoSSLModel:
├── SceneTransformerEncoder
├── MaskedTrajectoryPredictor (mask_ratio=0.15)
├── ContrastiveMapAgentHead (temp=0.1)
└── FuturePredictionHead (H=30 steps)

Training objectives:
├── Masked reconstruction loss
├── Contrastive alignment loss
└── Future prediction loss
```

### Usage
```bash
# Pre-train SSL encoder on Waymo data
python -m training.sft.train_waymo_ssl \
    --tfrecords /path/to/waymo/*.tfrecord \
    --output_dir out/ssl_waymo \
    --epochs 50 \
    --batch_size 32 \
    --lr 1e-4 \
    --mask_ratio 0.15

# Resume from checkpoint
python -m training.sft.train_waymo_ssl \
    --tfrecords /path/to/waymo/*.tfrecord \
    --resume out/ssl_waymo/best_model.pt
```

### Architecture
