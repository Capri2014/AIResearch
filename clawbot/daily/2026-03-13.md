# Daily Notes - 2026-03-13

## Pipeline PR #1: Unified CARLA Evaluation Pipeline v2

### Created: `training/eval/unified_carla_eval.py`
- **EvalConfig**: Configuration dataclass for unified evaluation
- **PolicyLoader**: Loads BC, RL, or SFT+Delta policy checkpoints
- **find_latest_bc_checkpoint()**: Auto-detect latest BC checkpoint
- **find_latest_rl_checkpoint()**: Auto-detect latest RL checkpoint
- **EpisodeMetrics**: Per-episode metrics (ADE, FDE, route completion, infractions)
- **compute_aggregate_metrics()**: Aggregate statistics across episodes

### Features:
- **Multi-policy support**: BC, RL, SFT+Delta policies
- **Auto-detection**: Automatically finds latest checkpoints
- **Multi-weather evaluation**: clear, cloudy, night, rain conditions
- **Multi-town support**: Evaluate across different CARLA towns
- **Comprehensive metrics**:
  - Waypoint metrics: ADE, FDE, speed error
  - ScenarioRunner metrics: route completion, collisions, offroad, red light
  - Success rate, duration, distance traveled
- **Weather-specific breakdowns**: Per-weather JSON files
- **ScenarioRunner integration**: Optional --use-srunner flag

### Architecture:
```
Policy Checkpoint (BC/RL/SFT_Delta)
    ↓
PolicyLoader.load()
    ↓
Policy Metadata + Device
    ↓
For each (weather, town, episode):
    ↓
run_carla_episode() → EpisodeMetrics
    ↓
compute_aggregate_metrics()
    ↓
metrics.json + per-weather JSON
```

### Usage:
```bash
# Dry-run (always works)
python3 -m training.eval.unified_carla_eval --dry-run

# Evaluate latest BC model (auto-detect)
python3 -m training.eval.unified_carla_eval \
    --weather clear,cloudy,night,rain \
    --num-episodes 10

# Evaluate specific BC checkpoint
python3 -m training.eval.unified_carla_eval \
    --checkpoint out/waypoint_bc/run_20260312_083423/final_checkpoint.pt \
    --policy-type bc

# Evaluate RL-refined policy
python3 -m training.eval.unified_carla_eval \
    --checkpoint out/ppo_sft_delta/run_20260312_213340/model.pt \
    --policy-type rl

# With ScenarioRunner integration
python3 -m training.eval.unified_carla_eval \
    --use-srunner \
    --srunner-root /path/to/scenario_runner \
    --srunner-suite smoke
```

### Output:
- `out/eval_unified/<run_id>/metrics.json` - Full results
- `out/eval_unified/<run_id>/config.json` - Configuration
- `out/eval_unified/<run_id>/weather_*.json` - Per-weather breakdown
