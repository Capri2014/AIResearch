## Tasks Completed (PR #4)

### 1. BEV Encoder Module (`sim/driving/carla_srunner/bev_encoder.py`)
- **BEVEncoder**: Unified encoder combining camera + LiDAR inputs
- **LidarToBEV**: Convert LiDAR point clouds to BEV grid representation
  - Height bin encoding (4 bins for elevation)
  - Intensity channel support
  - Projection layers for feature extraction
- **CameraToBEV**: Transform camera features to BEV via perspective projection
- **BEVEncoderConfig**: Configuration dataclass
- Supports fusion types: concat, attention, sum
- Factory function `create_bev_encoder()`
- `get_bev_image()` for visualization

### 2. Tests (`sim/driving/carla_srunner/test_bev_encoder.py`)

## Architecture
```
Camera Images → CameraToBEV → Camera BEV Features
LiDAR Points → LidarToBEV → LiDAR BEV Features
                              ↓
                      Feature Fusion
                              ↓
              BEV Features → Waypoint BC Model
```

## Usage
```python
from sim.driving.carla_srunner.bev_encoder import create_bev_encoder

encoder = create_bev_encoder(
    input_types=["camera", "lidar"],
    bev_resolution=0.5,
    bev_size=(200, 200),
    feature_dim=256
)

bev_features = encoder.encode(
    cameras=camera_images,
    lidar_points=lidar_pc,
)
```
