# Daily Notes: 2026-03-06

## Pipeline PR #1 (Today)

**Focus**: Temporal Waypoint Behavior Cloning

### What was done

Created new training script for temporal waypoint BC:
- **File**: `training/sft/train_temporal_waypoint_bc.py`
- **Feature**: Uses LSTM to aggregate temporal context from multiple frames
- **Architecture**:
  - `TemporalEncoder`: LSTM over per-frame CNN embeddings
  - `WaypointHead`: MLP mapping temporal embedding → waypoints
  - `TemporalEpisodesDataset`: Samples consecutive frames from episodes

### Key changes

1. **TemporalEncoder class**: LSTM-based temporal aggregation
   - Processes sequence of frames through CNN encoder
   - Uses final hidden state as sequence representation
   
2. **TemporalEpisodesDataset**: New dataset class
   - Samples consecutive frames from same episode
   - Creates temporal sequences of length `--sequence-length`
   - Uses last frame's waypoints as prediction target

3. **CLI arguments**:
   - `--sequence-length`: Number of frames in temporal context (default: 4)
   - `--hidden-dim`: LSTM hidden dimension (default: 256)
   - `--num-rnn-layers`: Number of LSTM layers (default: 2)

### Usage

```bash
python -m training.sft.train_temporal_waypoint_bc \
  --episodes-glob "out/episodes/**/*.json" \
  --sequence-length 4 \
  --batch-size 16 \
  --num-steps 200 \
  --out-dir out/temporal_waypoint_bc
```
