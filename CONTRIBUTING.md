# Contributing

This repo uses a simple PR workflow.

## Workflow
- **Default branch:** `main`
- Work happens on **feature branches** (`feature/*`).
- Open a PR and request review.

## ClawBot-assisted maintenance
This repository is maintained with help from **ClawBot (OpenClaw)**:
- ClawBot will propose changes as PRs.
- You (repo owner) review and merge.

## Guidelines
- Keep demos runnable (prefer small dependencies).
- Keep docs concise and verifiable (avoid hype; cite sources).
- Never commit secrets. Use env vars or mounted credential files.
